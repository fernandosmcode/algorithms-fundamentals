# 📊 Algoritmos – Fundamentos

Repositório criado para armazenar os exercícios, práticas e projetos realizados durante o curso **“Algoritmos”** do Curso em Vídeo (40 horas).

---

## 📌 Sobre o Repositório

Este repositório serve como registro do aprendizado em **lógica de programação e algoritmos**, reunindo exercícios desenvolvidos em **Portugol, Visualg** e outros ambientes de prática.

O objetivo é consolidar os conceitos fundamentais de algoritmos e servir como referência prática para futuros projetos.

---

## 📂 Organização

Os exercícios e projetos estão organizados de forma prática por linguagem ou tipo de atividade. Cada pasta contém seus próprios arquivos e, quando necessário, um README explicando os exercícios daquela área.

> Novos exercícios serão adicionados à medida que o curso avança e mais prática for realizada.

---

## 🎯 Propósito do Portfólio

Este repositório funciona como:

- Registro do aprendizado em lógica e algoritmos
- Referência de práticas realizadas no curso
- Demonstração de evolução em conceitos de programação
